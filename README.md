# Heart Disease Classification System

Dataset: UCI Heart Disease Dataset (loaded via ucimlrepo)

Models Implemented:
- Logistic Regression
- Decision Tree
- KNN
- Naive Bayes
- Random Forest
- XGBoost

Steps:
1. Place the 4 UCI raw dataset files in the project root.
2. Run: python train_models.py
3. Run: streamlit run app.py
